Les bases de l'html

  • Les balises

    • Les balises paires

      • Il s'agit de balises qui ont besoin absolument d'une balise ouvrante et d'une balise fermante pour fonctionner.

      <ul></ul> <li></li> <titre></titre> source : Jean Yard

    • Les balises orphelines

      • Il s'agit des balises qui n'ont pas besoin de balises de fermeture pour fonctionner. Pour plus de clareté, on parle habituellement plutôt de balise auto-fermentes. Elles servent le plus souvent à mettre des éléments à un endroit précis (souvent des images)

      Exemple : <br> <imput> <area> source : Jean Yard

  • Les attributs

    • Il s'agit des options des balises. Elles viennent compléter celles-ci pour donner des données supplémentaires. L'attribut se place après le nom de la balise et a le plus souvent une valeur pour pouvoir préciser la modification.

    source : Jean Yard

  • La différence entre inline et block

    • Block :crée de facon automatique un retour à la ligne avant et après. Cette facon de construire un site permet d'augmenter les possibilités de design pour celui-ci. Inline : Une balise de type inline se retrouve à l'intérieur d'une balise block. pas de retour à la ligne, le texte qui se trouve à l'intérieur s'écrit donc à la suite du texte précédent, sur la même ligne

  • Exemples de balises block

    • <p></p>
    • <footer></footer>
    • <article></article>

  • Exemples balises infline

    • <em></em>
    • <Strong></strong>
    • <mark></mark>
    • source : openclassrooms
